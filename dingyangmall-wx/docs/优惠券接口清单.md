# 优惠券/代金券 接口清单

## 一、数据库

执行 `sql/tb_coupon_info.sql` 创建表：

```bash
mysql -u user -p dingyangmall_ry < sql/tb_coupon_info.sql
```

或手动执行：

```sql
USE dingyangmall_ry;
-- 见 sql/tb_coupon_info.sql
```

---

## 二、管理端（PC 后台）

**Base path**：`/mall/coupon` 或 `/dev-api/mall/coupon`（代理时使用后者）

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/mall/coupon/page` | 分页查询优惠券（支持 userId、couponStatus、goodsId 等条件） | mall:coupon:index |
| GET | `/mall/coupon/{id}` | 按 id 查询单张券 | mall:coupon:get |
| POST | `/mall/coupon` | 新增券（通常由发放或订单自动生成） | mall:coupon:add |
| PUT | `/mall/coupon` | 修改券 | mall:coupon:edit |
| DELETE | `/mall/coupon/{id}` | 删除券 | mall:coupon:del |
| POST | `/mall/coupon/distribute` | 发放代金券 | mall:coupon:add |
| GET | `/mall/coupon/statistics` | 核销统计 | mall:coupon:index |

### 发放代金券请求体

```json
{
  "phone": "13800138000",
  "goodsId": "商品ID（goods_type=2 的商品）",
  "count": 1,
  "validityDays": 365
}
```

- `phone` 与 `userId` 二选一
- `goodsId` 必填
- `count` 默认 1，单次最多 100 张
- `validityDays` 默认 365

### 核销统计响应

```json
{
  "code": 200,
  "data": {
    "totalVerified": 100,
    "todayVerified": 5,
    "byDealer": 20
  }
}
```

`byDealer` 需传 `verifyDealerId` 参数时才返回。

## 三、移动端（小程序）

**Base path**：`/api/ma/coupon` 或 `/weixin/api/ma/coupon`（需带 Header `X-Wx-Token`）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/ma/coupon/my` | 我的优惠券/代金券列表 |

### 我的优惠券

- **参数**：`status` 可选，1=未使用，2=已使用，3=已过期；不传查全部
- **认证**：需登录（X-Wx-Token），按 openid 解析为 ums_member.id 后查券
- **响应**：`{ code, msg, data: [ 券列表 ] }`

**示例**：`GET /api/ma/coupon/my?status=1`

---

## 四、商家端（核销）

**Base path**：`/api/mall/merchant/scan`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/mall/merchant/scan/coupon/verify` | 核销商品券（扫券码） |

### 核销请求体

```json
{
  "couponCode": "券码（10位大写字母数字）"
}
```

**响应**：`{ code: 200, msg: "核销成功" }` 或 `{ code: 500, msg: "无效或已过期的商品券" }`

---

## 五、核心数据结构

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | 主键 |
| couponCode | String | 券码，唯一 |
| userId | Long | 持有用户（ums_member.id） |
| goodsId | String | 关联商品（goods_spu.id） |
| goodsName | String | 商品名称 |
| goodsPic | String | 商品图片 |
| integralPrice | Integer | 兑换积分 |
| validityStart | LocalDateTime | 有效期开始 |
| validityEnd | LocalDateTime | 有效期结束 |
| couponStatus | Integer | 1=未使用 2=已使用 3=已过期 |
| verifyTime | LocalDateTime | 核销时间 |
| verifyDealerId | Long | 核销经销商ID |
| verifyDealerName | String | 核销经销商名称 |

## 六、前端调用示例

**小程序 - 我的优惠券**：

```javascript
// utils/api.js 中
couponMyMa: (status) => {
  return request('/api/ma/coupon/my', 'get', status != null ? { status } : {}, false)
}

// 页面中
api.couponMyMa(1).then(res => {
  if (res.code === 200) {
    this.coupons = res.data || []
  }
})
```

**管理端 - 发放**：

```javascript
axios.post('/dev-api/mall/coupon/distribute', {
  phone: '13800138000',
  goodsId: '1234567890',
  count: 2,
  validityDays: 180
})
```
