# 管理端前端依赖的后端接口清单

当出现「系统繁忙」或接口不通时，可对照本清单检查后端是否已实现并正常返回。  
请求根路径以实际部署为准（如 `http://localhost:7500`），管理端需登录态（Token）。

---

## 一、抽奖活动（/mall/lottery）

| 功能     | 方法 | 路径                       | 说明 |
|----------|------|----------------------------|------|
| 获取配置 | GET  | `/mall/lottery/config`     | 返回抽奖开关、消耗积分、每日上限、未中奖概率、奖品列表等 |
| 保存配置 | POST | `/mall/lottery/config`     | Body: 同上结构，含 prizeList |
| 抽奖记录 | GET  | `/mall/lottery/record/page`| 分页，params: current, size |

---

## 二、积分规则（/integralrule）

| 功能     | 方法 | 路径                 |
|----------|------|----------------------|
| 分页     | GET  | `/integralrule/page` |
| 详情     | GET  | `/integralrule/{id}` |
| 新增     | POST | `/integralrule`      |
| 修改     | PUT  | `/integralrule`      |
| 删除     | DELETE | `/integralrule/{id}` |

**说明**：前端已对上述接口做容错处理，接口失败时不会白屏，仅会提示错误并可继续操作；后端按本清单实现或修复后即可联调。
